# Dashline

**STILL IN EARLY DEVELOPMENT STAGE!!!**

A Netease music player. Written in React+Shadcn and Tauri

## Development

```shell
# install dependencies
pnpm install

# start dev with tauri
pnpm start

# build for current platform
pnpm build

# build for android
pnpm tauri android build

```
